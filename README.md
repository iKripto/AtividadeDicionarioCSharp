*Descrição*

Este é um programa simples em C# desenvolvido como atividade de aprendizado.
Ele permite:

Registrar filmes

Avaliar filmes com notas de 0 a 10

Listar todos os filmes com suas avaliações

Calcular a média de avaliações de um filme

O programa utiliza dicionários para armazenar os filmes e suas listas de avaliações.
